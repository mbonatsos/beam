    Beam Settings

    Configure apps, quick links, search behavior, and local data from one place.

    App Library

    Add and manage searchable apps. Use tags for faster filtering, and optional action mode for dynamic paths.

    Use `{customerid}` style placeholders to append dynamic values from search input.

    Need to add many apps quickly? Use Data > Import from Chrome.

      Quick Links

      Add one-click links that always stay visible below search. Use them for tools you open multiple times a day.

      Leave icon URL empty to use an automatic site favicon.

        Experience and Behavior

        Control how Beam behaves while you search and navigate, including helper visibility and search sensitivity.

        Search Trigger Threshold Set how many characters are required before Beam shows search results.

        Suggested value: 3

        Lower values can reduce search performance on larger app lists.

        Web Search Providers

        Choose which web search providers appear in launcher results when your query has no direct app match.

        Tip: press Tab on an empty search box to jump into quick web search results.

        Backup and Restore

        Manage your local Beam setup. Export a backup, restore from JSON, import from Chrome, or reset to defaults.

        Partial Import from JSON

        Keep your current setup and add only new apps from a JSON file.

        Import from Chrome

        Bring in bookmarks as apps. You can grant permission when prompted.

        Export

        Download your current setup as a JSON file.

        Restore from JSON

        Replace your current local setup with a backup file.

        Reset

        Clear local changes and go back to your default setup.
